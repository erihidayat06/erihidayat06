Website My Makeup

Website MyMakeup ini saya buat untuk tugas web programing II yang sudah menggunakan framework Laravel.

Website ini adalah website yang dimana digunakan untuk meningkatkan usaha para Makeup Artis (MUA) agar mudah di jumpai para kostumer. kostumer juga tidak perlu lagi datang langsung ke salon untuk memesan Makeup artis.

Selain untuk tugas kuliah website ini juga saya kembangin lagi untuk mengikuti lomba Krenova 2023 di Kota Tegal dan alhamdulilah mendapatkan juara harapan III bersama tim saya.
